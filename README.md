## 菜谱

- 卤肉饭
1. 五花肉切细条，洋葱，葱，蒜切细
2. 五花肉炒致金黄出油
3. 放入洋葱炒出香味
4. 放入葱炒出香味
5. 放入蒜炒出香味
6. 放入糖炒到颜色稍微变深
7. 沿锅边导入酱油翻炒
8. 倒入水，五香粉，白胡椒粉，放入鹌鹑蛋以及豆干
9. 水烧开后，小火慢慢熬制
